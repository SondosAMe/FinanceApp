# Quick Start: Add to iPhone Home Screen

## Easiest Method (2 Minutes)

### Step 1: Host Your App (Choose One)

**Option A: Netlify Drop (Easiest - No Account Needed)**
1. Go to: https://app.netlify.com/drop
2. Drag your entire `FinanceApp` folder
3. Copy the URL you get (e.g., `https://your-app.netlify.app`)

**Option B: GitHub Pages (Free Account Needed)**
1. Create GitHub account
2. Create new repository
3. Upload all files
4. Settings > Pages > Enable
5. Get URL: `https://yourusername.github.io/repo-name`

### Step 2: Add to iPhone

1. **Open Safari** on your iPhone (important: must be Safari, not Chrome)
2. **Type or paste** your app URL in the address bar
3. **Wait** for the page to load completely
4. **Tap the Share button** (square with arrow â†‘) at the bottom
5. **Scroll down** and tap **"Add to Home Screen"**

## Troubleshooting

**"Add to Home Screen" not showing?**
- Make sure you're using Safari (not Chrome)
- Refresh the page
- Make sure URL starts with http:// or https://

**Charts not working?**
- Need internet connection (for Chart.js)
- Try refreshing

**Data not saving?**
- Make sure you're using a URL (not file://)
- Check Safari settings
